# Aethelgard Botanical Farms Website

A premium, fully responsive luxury botanical farm website developed as part of the Frontend Web Development Internship at Cognifyz IT Solutions. 

## 🚀 Features
- **Elegant Luxury UI:** Premium dark-themed design tailored for a high-end botanical farm experience.
- **Fully Responsive:** Completely optimized for Mobile, Tablet, and Desktop screens.
- **Interactive Sections:** Beautiful hero section, detailed farm offerings, heritage story, and a seamless contact form.
- **Modern Tech Stack:** Built using React, Vite, TypeScript, and Tailwind CSS for lightning-fast performance.

## 🛠️ Tech Stack Used
- **Frontend:** React.js (TypeScript)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Build Tool:** Vite
